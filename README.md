# Windows 11 图片转 Word 工具 (Jpg2Word)

这是一个基于 Python 开发的现代化图片文字识别 (OCR) 并生成 Word 文档的桌面工具，特别针对 Windows 11 中文版进行了优化，支持多国语言识别及排版保持。

## 🌟 核心功能

- **多语言识别**：支持 **中文 (简体)**、**英文** 以及 **印尼语**。
- **高精度 OCR**：采用 `EasyOCR` 识别引擎，在 CPU 模式下运行极其稳定。
- **格式排版保持**：智能行检测与动态空格模拟，最大程度还原图片中的段落与横向间距。
- **自动保存**：识别结果将自动保存至**源图片所在文件夹**，并以 `图片名_识别结果.docx` 命名。
- **纸张大小调节**：支持 A4、A3 和 Letter 纸张规格设置。
- **现代化 UI**：使用 `customtkinter` 打造，完美契合 Windows 11 视觉风格。
- **自动打开**：转换完成后可选择一键打开生成的 Word 文档。

## 🛠️ 安装说明

### 1. 环境要求
- **操作系统**：Windows 10/11
- **Python 版本**：3.8 或更高版本 (推荐使用 3.10+)

### 2. 获取源码
```bash
git clone https://github.com/syscca/Jpg2Word.git
cd Jpg2Word
```

### 3. 安装依赖项
在终端 (CMD 或 PowerShell) 中运行以下命令安装必要的库：
```bash
pip install -r requirements.txt
```

### 4. 运行程序
```bash
python jpg2word.py
```

## 📝 使用指南

1. **选择图片**：点击中间显眼的“选择图片文件”按钮，支持多选。
2. **设置参数**：
   - 在左侧侧边栏选择正确的“识别语言”。
   - 根据需要设置“纸张大小”。
   - “使用 GPU”选项默认关闭，将使用 CPU 稳定运行。
3. **开始转换**：点击“开始转换”。
   - **首次运行提示**：如果是第一次运行，程序会自动下载约 100MB 的 OCR 识别模型，请确保网络通畅。
4. **查看结果**：转换完成后，点击提示框中的“是”即可立即打开位于图片目录下的 Word 文档。

## ⚠️ 注意事项

- **CPU 模式提示**：运行过程中如果终端提示 `Using CPU. Note: This module is much faster with a GPU.` 属于正常现象，程序已针对 CPU 稳定性进行了优化。
- **Word 兼容性**：生成的文档采用标准的 `.docx` 格式，默认字体为“宋体 (SimSun)”，兼容 Office 2013 及以上版本。

## 📦 依赖库列表
- `easyocr` (OCR 引擎)
- `python-docx` (Word 生成)
- `customtkinter` (GUI 界面)
- `Pillow` (图片处理)
- `numpy` (数据计算)
